# Prompt-Extractor 学习指南
## 如何通过结构化分析学习提示词写作技巧

**创建日期**: 2026-01-04
**目标用户**: 想要提升AI图像提示词写作能力的创作者

---

## 🎯 核心学习价值

prompt-extractor 不是简单的文本解析工具，而是一个 **提示词写作教练**。

### 它能教会你什么？

1. **结构思维** - 如何组织一个复杂提示词
2. **模块化写作** - 如何拆解和重组元素
3. **质量标准** - 什么是好的提示词
4. **高级技巧** - 专业级的写作策略
5. **避坑指南** - 常见错误和改进方向

---

## 📚 5种学习场景

### 场景1: 解构学习 - "这个提示词为什么好？"

**学习目标**: 理解优秀提示词的组织结构

#### 示例：学习 "Ethereal Cyber-Deity" 提示词

**原始提示词** (180词):
```
An ethereal deity composed of intricate white translucent optical fibers...
Shot on 105mm Macro lens, f/1.8, 8k resolution, hyper-realistic...
```

**通过 prompt-extractor 分析后**，你会看到:

```markdown
## 结构分析

7层金字塔式组织:
1. Subject (主体) - 定义"是什么"
2. Space (空间) - 定义"在哪里"
3. Action (动作) - 定义"在做什么"
4. Fashion (装置) - 定义"穿戴什么"
5. Color (色彩) - 定义"什么颜色"
6. Details (细节) - 定义"特写什么"
7. Style (风格) - 定义"怎么渲染"
```

**学到的技巧**:
- ✅ **递进式组织** - 从宏观到微观，层层深入
- ✅ **明确的模块边界** - 每一层职责清晰
- ✅ **参数化表达** - 使用具体数值（45°, f/1.8, 5cm）而非模糊词

**立即可用**:
你下次写提示词时，可以套用这个7层结构：
```
我的提示词:
1. Subject: "一个水晶雕塑"
2. Space: "放在黑色丝绒上"
3. Action: "被光线穿透"
4. Fashion: "无"
5. Color: "透明+彩虹折射"
6. Details: "内部气泡、边缘锐利"
7. Style: "微距摄影，Octane渲染"
```

### 场景2: 对比学习 - "不同风格的差异在哪？"

**学习目标**: 通过对比理解风格差异

#### 示例：对比两个人像提示词

**提示词A** (清纯少女风):
```json
{
  "color_scheme": {
    "palette": ["soft pink", "cream white", "gentle peach"],
    "saturation": "低饱和柔和色调"
  },
  "mood_atmosphere": "innocent, gentle, youthful",
  "lighting": "soft diffused golden hour light"
}
```

**提示词B** (赛博朋克风):
```json
{
  "color_scheme": {
    "palette": ["neon pink", "electric blue", "deep purple"],
    "saturation": "高饱和霓虹色"
  },
  "mood_atmosphere": "edgy, futuristic, mysterious",
  "lighting": "harsh neon rim lighting"
}
```

**学到的技巧**:
- 🎨 **色彩=情绪** - 柔和色→清纯，霓虹色→前卫
- 💡 **光线=氛围** - 柔光→温暖，硬光→冷酷
- 📝 **词汇选择** - gentle vs edgy, soft vs harsh

**展示方式**:
```
┌─────────────────────────────────────────┐
│         风格对比速查表                      │
├───────────┬─────────────┬───────────────┤
│ 风格维度     │ 清纯少女风       │ 赛博朋克风        │
├───────────┼─────────────┼───────────────┤
│ 主色调      │ 粉/白/桃        │ 霓虹粉/电蓝/紫     │
│ 饱和度      │ 低            │ 高             │
│ 光线类型     │ 柔和漫射光       │ 硬边霓虹光        │
│ 情绪关键词    │ innocent     │ edgy          │
│ 相机设置     │ f/1.8 柔焦    │ f/4 锐利        │
└───────────┴─────────────┴───────────────┘
```

### 场景3: 模板学习 - "这个技巧如何复用？"

**学习目标**: 提取可复用的写作模板

#### 示例：学习 "Cold-Warm Color Opposition" 策略

**从分析报告中提取**:
```markdown
## 高价值可复用模板

### Template: Cold-Warm Color Opposition

**结构**:
{subject}, Color Palette: {body zone} = {cool colors},
{focal object} = {warm colors},
Lighting from {focal object} illuminating {subject}

**原理**:
- 冷色环境 → 营造距离感、神秘感
- 暖色焦点 → 吸引注意力、制造对比
- 光源来自焦点 → 增强戏剧性

**应用场景**:
- 人像摄影（冷环境 + 暖光源）
- 产品摄影（冷背景 + 暖产品）
- 概念艺术（情绪对比）
```

**学习过程**:
1. **看到技巧** - 在分析报告中识别高分模板
2. **理解原理** - 为什么这样做有效？
3. **练习应用** - 套用到自己的创作

**练习示例**:
```
原模板:
"Entity, Color: Body=cyan/teal, Cube=pink/amber,
Lighting from cube"

我的应用:
"Crystal sorceress, Color: Body=ice blue/silver,
Magic orb=ruby red/gold, Lighting from orb"
```

**学到的技巧**:
- 🎨 **色彩对立原理** - 冷暖对比 = 视觉冲击
- 💡 **剧情光源** - 场景内物体发光 > 外部打光
- 📐 **模板化思维** - 变量替换，结构不变

### 场景5: 避坑学习 - "常见错误如何避免？"

**学习目标**: 从AI生成挑战中学习注意事项

#### 示例：学习AI生成的难点

**从分析报告中看到**:
```markdown
## AI生成挑战点

| 挑战 | 难度 | 说明 |
|------|------|------|
| 纤维网络细节 | ⭐⭐⭐⭐⭐ | AI可能简化为规则纹理 |
| 次表面散射控制 | ⭐⭐⭐⭐ | 手指透光效果需高级渲染 |
| 棱镜折射 | ⭐⭐⭐⭐⭐ | 彩虹色散物理准确性难保证 |
| 微型城市细节 | ⭐⭐⭐⭐⭐ | 5cm³内的建筑群需超高分辨率 |
```

**避坑指南**:
```
❌ 错误写法:
"a complex fibrous network"
→ AI会生成简单的网格纹理

✅ 正确写法:
"millions of overlapping white filaments, chaotic organic structure,
similar to spider silk macro photography, each fiber clearly visible"
→ 强调"混沌"、"百万级"、"每根可见"
```

**学到的技巧**:
- 🚨 **预判AI弱点** - 复杂细节需要强调
- 📝 **强化描述** - 用"millions", "each fiber visible"等强化词
- 🔍 **添加参考** - "similar to spider silk" 帮助AI理解

## 🎓 完整学习路径

### 新手路径 (0-10小时)

**第1步: 解构分析** (2小时)
- 用 prompt-extractor 分析 3-5 个优秀提示词
- 重点看 `structure_type` 和 `extracted_patterns`
- 理解"7层结构"、"3层景深"等基本概念

**第2步: 模仿练习** (3小时)
- 选择一个喜欢的提示词
- 套用它的结构，替换主体
- 对比原作和你的版本

**第3步: 模板应用** (5小时)
- 从 `reusable_templates` 中选择 2-3 个模板
- 在自己的创作中应用
- 观察效果差异

---

### 进阶路径 (10-50小时)

**第4步: 对比分析** (10小时)
- 分析同一主题的不同风格提示词
- 提取风格差异的关键参数
- 建立自己的"风格速查表"

**第5步: 技术深化** (15小时)
- 系统学习 `technical_parameters` 部分
- 理解相机参数、渲染引擎、光学效果
- 建立"技术参数工具箱"

**第6步: 创新实验** (25小时)
- 尝试组合不同模板
- 创造新的色彩策略、构图方法
- 分析自己的创新是否有效

---

### 大师路径 (50+小时)

**第7步: 建立理论体系**
- 总结自己的写作原则
- 建立个人的"提示词质量标准"
- 形成独特的写作风格

**第8步: 教学输出**
- 分享你的学习心得
- 创建自己的提示词模板库
- 帮助其他人学习
